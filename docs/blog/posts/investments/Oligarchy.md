# Oligarchy

本文介绍寡头市场和寡头行业的投资机会

<!-- more -->

垄断市场、寡头市场：门槛高，利润高，规模效应。

完全竞争压价走量，垄断寡头躺着赚钱

## 垄断的发生

- 自然垄断：以前期投入为主的行业，如电力、水务，当有新竞争者出现的时候，可以用极低的价格将对手轻松挤出局
- 高门槛：需要授权、高研发投入、规模效应带来的低成本优势

## 寡头市场

寡头市场中，只有几个卖方，寡头们非常容易形成合谋抬高价格，减少竞争。寡头们合谋的形式有：

| 形式 | 描述 | 例子 |
| --- | --- | --- |
| 寡头之间形成默契 | 抬高价格会丢失市场，降低价格会打价格战 | Cocacola & Pepsi |
| 寡头之间合谋 | 寡头之间合谋控制市场价格和供给 | OPEC |

而垄断组织的形式有：

| 垄断组织形式 | 描述 | 例子 |
| --- | --- | --- |
| 卡特尔 Cartel | 保持互相独立，签订协议联合 | OPEC |
| 辛迪加 Syndicate | 企业名义上独立，经营上统一采购统一销售 | 日本农协，连锁加盟店 |
| 托拉斯 Trust| 多家企业被一家伞公司控制，从而对整个行业进行垄断 | 中国烟草、中国神华、中国船舶 |
| 康采恩 Concern | 跨行业垄断组织 | 韩国三星、日本三菱 |

## “垄断竞争”：有产品差异化的完全竞争

“垄断竞争”：差异化竞争，实际上多描述完全竞争行业。
